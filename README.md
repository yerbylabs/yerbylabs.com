# yerbylabs.com

Static website for Yerby Labs, hosted on Cloudflare Pages.

## Structure

```
index.html                  — Main landing page
ironbeak/privacy.html       — IronBeak privacy policy
_redirects                  — Cloudflare Pages redirect rules
favicon.png                 — Site favicon
```

## Deployment

This repo is connected to Cloudflare Pages for automatic deployment on every push to `main`.

- **Production URL:** https://yerbylabs.com
- **Privacy policy:** https://yerbylabs.com/ironbeak/privacy.html
